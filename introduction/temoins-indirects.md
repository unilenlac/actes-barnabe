# Formes courtes se rattachant à BHL 985 qui ne méritent pas d'édition à part.

Certaines formes courtes sont essentiellement des versions raccourcies des formes longues, par élagage de certaines parties. C'est pourquoi nous ne proposons pas d'édition spécifique de ces formes. Pour autant, nous les avons intégré dans notre travail de collation. L'édition papier ne présente pas leurs variantes dans l'apparat, mais l'édition numérique permet de les faires apparaîtres sur demande.
